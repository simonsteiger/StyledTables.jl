# Export

A `StyledTable` can be exported as HTML, docx, LaTeX, and Typst.

The necessary functions are included in [SummaryTables.jl](https://pumasai.github.io/SummaryTables.jl/stable), see its documentation on [Renderers](https://pumasai.github.io/SummaryTables.jl/stable/reference/renderers) for further details.
